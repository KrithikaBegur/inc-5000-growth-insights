# inc-5000-growth-insights
This project analyzes the 2019 INC 5000 dataset to highlight where high-growth companies are emerging across the United States and which industries are expanding the fastest. Using Power BI, the report explores national and state-level trends, compares revenue and workforce patterns, and identifies the sectors driving the strongest growth momentum. The dashboard provides a clear, interactive view of how industry performance, geographic distribution, and company scale intersect-offering practical insights into the landscape of high-growth businesses in the U.S.
